## 实验1：构建更公平的强基线（AnalogSeeker + Ours）

- **状态**：已完成  
- **核心发现**：在ReasoningV-7B上，统一优化策略显著提升，全面超越或达到AnalogSeeker基准。  
- **主要指标（优化前 → 优化后）**  
  - LDO：46.0% → **81.6%**（+35.6%，+77.4%）  
  - Comparator：60.0% → **76.0%**（+16.0%，+26.7%）  
  - Bandgap：58.0% → **70.0%**（+12.0%，+20.7%）  
  - TQA：85.0% → **93.32%**（+8.32%，+9.8%）  
  - Caption：32.5% → **61.27%**（+28.77%，+88.5%）  
  - Opamp：33.3% → **58.33%**（+25.0%，+75.1%）  
  - **总体**：79.01% → **86.66%**（+7.65%，+9.68%）
- **结论**：Few-shot与专家提示、确定性参数、多策略混合等组合带来大幅收益，所有6个任务均达到或超越AnalogSeeker基准。

## 路由与错误分析（实验3、实验5）

- **状态**：已完成（Router有正面效果）  
- **核心结果**：分类准确率 **85.50%**（171/200），错误率14.50%。  
- **关键发现**：  
  - 计算类问题最易混淆：24个计算题中20个被误分（83.3%）。  
  - 分类错误会导致策略失配，证明“对症下药”必要性。  
  - 改进方向：加强计算题识别（关键词/数值/公式特征），对边界样本可用多策略融合或更细规则。  
- **产出文件**：`experiment3_router_sensitivity_results.json`，`experiment3_router_sensitivity_report.md`

## 消融与策略洞察

- **消融实验报告**：Few-shot是最强单项贡献；策略组合效果叠加；确定性参数提供稳定基线；专家角色为Few-shot提供支撑。  
- **两次优化完整总结 / 按题型分类 / TQA按难度分类**：  
  - 示例数最优点：LDO≈3例，Comparator≈2例，Caption≈8例。  
  - TQA多策略混合（Answer precisely / Analyze carefully / Circuit Expert）达到 93.32%。  
  - 难度分布：Undergraduate/Graduate高精度，Unknown提升显著。  

## 模型特性与对比

- **ReasoningV与Analogseeker优化对比完整总结**：同策略下，ReasoningV提升幅度更大；Analogseeker起点高但提升有限。  
- **模型特性与优化策略匹配度深度分析**：  
  - ReasoningV：通用推理，需要明确任务指导、Few-shot与专家角色激活知识。  
  - Analogseeker：领域微调，基线强，对提示词依赖小，但策略提升空间有限。  
- **路由机制实现（论文格式优化总结）**：规则+语义嵌入混合；五类问题映射策略；TQA经路由+多策略由85%提至93.32%；需加强计算题识别与边界样本多策略融合。
